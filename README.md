# 概要
水分摂取量を管理するアプリ

# 機能一覧
- 水分摂取をml単位で記録する
- 本日摂取した水分量をml単位で確認できる
- 1日単位で水分摂取量と摂取時間を確認できる
- グラフで水分摂取量を確認できる(1週間単位)
- カレンダーで水分摂取量を確認できる(1ヶ月単位)

# 主な使用技術
- gin-gonic(v1.8.2)
- golang-jwt(v4.5.0)
- godotenv(v1.5.1)
- crypto(v0.22.0)
- Render(デプロイ機能)
